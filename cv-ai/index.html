🔑 Setup AI Provider

🎉 HOÀN TOÀN MIỄN PHÍ với Google account! Free 1500 requests/ngày, không cần thẻ tín dụng.

Cách lấy key (2 phút):

  1. Vào aistudio.google.com/app/apikey
  2. Đăng nhập Google → bấm Create API Key
  3. Chọn project (mặc định) → Copy key
  4. Paste vào ô dưới + Lưu
📋 KHÔNG CẦN KEY GÌ! Hệ thống tạo sẵn prompt — anh chỉ copy + paste vào ChatGPT/Claude.ai web (tài khoản free).

Workflow:

  1. Upload CV ở trang chính → hệ thống trích xuất text
  2. Bấm 🤖 Tạo prompt → app generate prompt sẵn (đã ghép CV vào)
  3. Copy prompt → mở chatgpt.com hoặc claude.ai → paste → gửi
  4. AI trả về JSON → copy lại → paste vào ô "Kết quả AI" trong app
  5. App tự parse và hiển thị đẹp

Phù hợp khi: chỉ phỏng vấn ít người, không muốn setup gì, đã có sẵn tài khoản ChatGPT/Claude.ai.

Cần 1 OpenAI API Key (~$0.0001 / CV ≈ miễn phí).

Cách lấy:

  1. platform.openai.com/api-keys
  2. Đăng ký + Create new secret key (có $5 free credit)
  3. Copy key dạng sk-... → paste dưới

Claude AI gọi qua Apps Script proxy (do Anthropic chặn CORS từ browser).
Anthropic API Key được lưu trong Apps Script (secure hơn OpenAI direct).

Cách setup:

  1. Đã setup Google Sheet ở /setup-database/ (cần làm trước)
  2. Lấy key tại console.anthropic.com/settings/keys ($5 free)
  3. Mở Apps Script → tìm const ANTHROPIC_API_KEY = ''; → paste key
  4. Deploy → New version → Deploy
  5. Quay lại đây → bấm Lưu (không cần điền key dưới)
ℹ️ Phải hoàn tất Bước 2 ở setup-database để Apps Script biết Anthropic Key.

🔒 OpenAI key lưu localStorage máy này. Anthropic key lưu trong Apps Script (an toàn hơn).
