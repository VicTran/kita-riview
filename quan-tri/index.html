📖 Hướng dẫn deploy file mới

  1. Bấm 💾 Download config.js ở khung trên → file lưu vào ~/Downloads
  2. Mở Finder/Explorer → thư mục Mô tả công việc/kita-riview/
  3. Kéo file config.js mới đè lên file cũ (hoặc copy + replace)
  4. Mở GitHub Desktop → thấy config.js đã thay đổi
  5. Bấm Commit to main → Push origin
  6. Đợi 30-60 giây → vào phongvanmkt.kitagroupvn.com/ bằng máy/trình duyệt khác → ✅ Auto load config mới
🎉 Từ giờ trở đi: HR mở bất kỳ trang nào trên bất kỳ máy nào — không cần setup gì, dùng được ngay.
