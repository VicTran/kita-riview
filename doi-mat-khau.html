🔑 Đổi mật khẩu hệ thống phỏng vấn

Tạo SHA-256 hash để cập nhật mật khẩu trong file index.html

BƯỚC 1: Nhập mật khẩu mới

// SHA-256 hash của mật khẩu mới:

BƯỚC 2: Cập nhật file index.html

1. Mở file index.html bằng VS Code / Notepad++

2. Tìm dòng: const PASSWORD_HASH = '...';

3. Thay hash cũ bằng hash mới vừa copy

4. Lưu file → GitHub Desktop → Commit + Push

5. Đợi 30-60s → mật khẩu mới có hiệu lực

⚠️ LƯU Ý BẢO MẬT

• Mật khẩu hiện tại: Kita@2026 - hãy đổi sau lần dùng đầu

• Mật khẩu mạnh: ≥8 ký tự, có chữ hoa + số + ký tự đặc biệt

• KHÔNG lưu mật khẩu trong file plain text

• Chia sẻ qua kênh bảo mật (1Password / miệng)
