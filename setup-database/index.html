☁️ Setup Google Sheet Database

Hướng dẫn kết nối hệ thống phỏng vấn với Google Sheet — không cần code

📋 Tại sao cần Google Sheet?

Hiện tại mỗi lần lưu, hệ thống tạo 1 file JSON tải về máy. Khó xem lại, dễ thất lạc, không sort/filter được. Sau khi setup Google Sheet:

  • ✅ Mọi hồ sơ tự động lưu lên 1 Google Sheet duy nhất
  • ✅ Xem tất cả ứng viên trong 1 bảng — sort theo điểm, lọc theo ngày
  • ✅ Export Excel, share cho team, làm báo cáo dễ dàng
  • ✅ Backup tự động bởi Google Drive
  • ✅ Tải lại bất kỳ hồ sơ nào để xem chi tiết / chỉnh sửa

🚀 BƯỚC 1: Tạo Google Sheet mới

1

Truy cập sheets.new

Hoặc vào Google Drive → New → Google Sheets

2

Đặt tên Sheet: KITA Interview Database

Đặt tên ô tab (sheet con) ở dưới là PhongVan (quan trọng - không được sai)

3

Copy header row vào dòng 1

Copy đoạn dưới và paste vào ô A1 của sheet (Google Sheet sẽ tự tách cột):

Timestamp Tên ứng viên Vị trí Ngày PV HR/PV viên Vòng SĐT/Email Nguồn Scores (JSON) Notes (JSON) QA (JSON) Decision (JSON) HR Decision (JSON) Điểm TB Đề xuất Full JSON

⚙️ BƯỚC 2: Tạo Apps Script (Backend)

1

Trong Google Sheet vừa tạo: Extensions → Apps Script

Một tab mới mở ra với editor code

2

Xoá toàn bộ code mặc định, paste code dưới đây:

// ============== KITA INTERVIEW DATABASE + AI ============== // Apps Script Web App - Kết nối hệ thống phỏng vấn với Google Sheet + Claude AI // Phiên bản: 2.0 | Tác giả: KITA Interview System const SHEET_NAME = 'PhongVan'; const API_KEY = 'KITA-2026-SECRET'; // ⚠️ Đổi key này thành chuỗi bí mật của bạn // ⚠️ Nếu muốn dùng Claude (Anthropic) cho AI đánh giá CV, paste key vào đây // Lấy key tại: https://console.anthropic.com/settings/keys // Để trống nếu chỉ dùng OpenAI từ trang cv-ai const ANTHROPIC_API_KEY = ''; // Ví dụ: 'sk-ant-api03-...' // ============== POST: Lưu hồ sơ HOẶC phân tích CV bằng Claude ============== function doPost(e) { try { const data = JSON.parse(e.postData.contents); if (data.apiKey !== API_KEY) { return jsonResponse({ ok: false, error: 'Sai API key' }); } // Action: phân tích CV bằng Claude if (data.action === 'analyze_cv') { return analyzeCVWithClaude(data); } // Action mặc định: lưu hồ sơ vào sheet const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); if (!sheet) { return jsonResponse({ ok: false, error: 'Không tìm thấy sheet "' + SHEET_NAME + '"' }); } const scores = Object.values(data.scores || {}).filter(s => typeof s === 'number'); const avg = scores.length ? (scores.reduce((a,b) => a+b, 0) / scores.length).toFixed(2) : ''; let rec = ''; if (avg >= 4) rec = '✅ PASS'; else if (avg >= 3) rec = '🟡 CÂN NHẮC'; else if (scores.length > 0) rec = '❌ LOẠI'; const candName = (data.candidate || {}).name || ''; const exportType = data.exportType || 'all'; const existingRow = findExistingRow(sheet, candName, exportType); const rowData = [ new Date(), candName, (data.candidate || {}).position || '', (data.candidate || {}).date || '', (data.candidate || {}).interviewer || '', exportType, (data.candidate || {}).contact || '', (data.candidate || {}).source || '', JSON.stringify(data.scores || {}), JSON.stringify(data.notes || {}), JSON.stringify(data.qa || []), JSON.stringify(data.decision || {}), JSON.stringify(data.hrDecision || {}), avg, rec, JSON.stringify(data) ]; if (existingRow > 0) { sheet.getRange(existingRow, 1, 1, rowData.length).setValues([rowData]); return jsonResponse({ ok: true, action: 'update', row: existingRow }); } else { sheet.appendRow(rowData); return jsonResponse({ ok: true, action: 'insert', row: sheet.getLastRow() }); } } catch (err) { return jsonResponse({ ok: false, error: err.message }); } } // ============== Phân tích CV bằng Claude (Anthropic) ============== function analyzeCVWithClaude(data) { if (!ANTHROPIC_API_KEY || !ANTHROPIC_API_KEY.startsWith('sk-ant')) { return jsonResponse({ ok: false, error: 'ANTHROPIC_API_KEY chưa được cấu hình trong Apps Script. Xem hướng dẫn ở /setup-database/' }); } try { const res = UrlFetchApp.fetch('https://api.anthropic.com/v1/messages', { method: 'post', contentType: 'application/json', headers: { 'x-api-key': ANTHROPIC_API_KEY, 'anthropic-version': '2023-06-01' }, payload: JSON.stringify({ model: data.model || 'claude-haiku-4-5-20251001', max_tokens: 2000, system: data.systemPrompt, messages: [{ role: 'user', content: 'CV ỨNG VIÊN:\n\n' + (data.cvText || '').slice(0, 12000) }] }), muteHttpExceptions: true }); const code = res.getResponseCode(); const text = res.getContentText(); if (code !== 200) { return jsonResponse({ ok: false, error: 'Claude API ' + code + ': ' + text.slice(0, 300) }); } const result = JSON.parse(text); const content = result.content[0].text; // Try parse JSON từ response của Claude let analysis; try { // Claude đôi khi wrap JSON trong ```json ... ``` const jsonMatch = content.match(/```json\s*([\s\S]*?)\s*```/) || content.match(/\{[\s\S]*\}/); const jsonStr = jsonMatch ? (jsonMatch[1] || jsonMatch[0]) : content; analysis = JSON.parse(jsonStr); } catch (e) { return jsonResponse({ ok: false, error: 'Claude trả về không phải JSON: ' + content.slice(0, 200) }); } return jsonResponse({ ok: true, analysis: analysis, model: data.model }); } catch (err) { return jsonResponse({ ok: false, error: 'Lỗi: ' + err.message }); } } // ============== GET: Tải danh sách hồ sơ ============== function doGet(e) { try { if (e.parameter.apiKey !== API_KEY) { return jsonResponse({ ok: false, error: 'Sai API key' }); } const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); const data = sheet.getDataRange().getValues(); if (data.length < 2) return jsonResponse({ ok: true, data: [] }); const headers = data[0]; const rows = data.slice(1); let result = rows.map((row, idx) => { const obj = { _row: idx + 2 }; headers.forEach((h, i) => obj[h] = row[i]); return obj; }).filter(r => r['Tên ứng viên']); // Lọc theo tên nếu có if (e.parameter.search) { const q = e.parameter.search.toLowerCase(); result = result.filter(r => ((r['Tên ứng viên'] || '') + '').toLowerCase().includes(q) || ((r['SĐT/Email'] || '') + '').toLowerCase().includes(q) ); } // Trả về 50 dòng mới nhất result.sort((a, b) => new Date(b.Timestamp) - new Date(a.Timestamp)); if (result.length > 50) result = result.slice(0, 50); return jsonResponse({ ok: true, data: result, total: result.length }); } catch (err) { return jsonResponse({ ok: false, error: err.message }); } } function findExistingRow(sheet, name, exportType) { if (!name) return 0; const data = sheet.getDataRange().getValues(); for (let i = 1; i < data.length; i++) { if (data[i][1] === name && data[i][5] === exportType) return i + 1; } return 0; } function jsonResponse(obj) { return ContentService.createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); }

🤖 (Tuỳ chọn) Bật Claude AI cho /cv-ai/

Mặc định trang /cv-ai/ dùng OpenAI gọi trực tiếp từ browser. Nếu muốn dùng Claude (Anthropic) chất lượng cao hơn, cần bật route qua Apps Script:

1

Lấy Anthropic API Key

Vào console.anthropic.com/settings/keys → Create Key → Copy

Key có dạng: sk-ant-api03-... (~$5 free credit cho người mới)

2

Mở Apps Script đã setup ở Bước 2 ở trên

Tìm dòng: const ANTHROPIC_API_KEY = '';

Thay bằng: const ANTHROPIC_API_KEY = 'sk-ant-api03-...'; (paste key vừa copy)

3

Lưu file (Ctrl+S) → Deploy → Manage deployments → Edit (icon bút chì)

Chọn "New version" → bấm Deploy → cấp quyền nếu Google hỏi

URL Web App KHÔNG đổi — chỉ đợi 30s là code mới có hiệu lực

4

Sang trang /cv-ai/ → bấm 🔑 API Key → chọn provider "Claude"

Không cần điền thêm key (key đã trong Apps Script). Bấm Lưu là xong.

✅ Sau bước này, mọi phân tích CV sẽ chạy qua Claude Haiku 4.5 — nhanh, chất lượng cao, ~$0.001 / CV (gần miễn phí).
